Photoshop ကို အသုံးပြုပြီး Web Template များကို ဖန်တီးချင်သူများ၊ မိမိကိုယ်တိုင်ဖန်တီးထားသော Web Template များကို HTML, CSS, JavaScript, jQuery အသုံးပြုကာ Web Browser များတွင် မည်ကဲ့သို့ အလုပ်လုပ်သည်ကိုလေ့လာချင်သူများ၊ Web Designer, UI/UX Designer နှင့် Front-End Designer ဖြစ်ချင်သူများ၊ တက်ရောက်သင့်သော သင်တန်းဖြစ်သည်။
Photoshop ကို အသုံးပြုပြီး Grid System ဖြင့် Web Template များဖန်တီးတည်ဆောက်ပုံများ၊ UI/UX သဘောတရားများကို အခြေခံမှစပြီး သင်ကြားပေးသော သင်တန်းဖြစ်သည်။ HTML, CSS element များကို အခြေခံမှ စတင်သင်ကြားပေးမှာဖြစ်ပြီး CSS framework တစ်ခုဖြစ်သော Bootstrap ကို အသုံးပြုကာ Responsive website တစ်ခုကို တည်ဆောက်နိုင်သည်အထိ သင်ကြားပေးသောသင်တန်းဖြစ်သည်။ သင်တန်းကာလအတွင်းတွင်လည်း UI Template များ၊ Website Template Design များကို ဆရာနှင့်အတူ ဖန်တီးသွားမည်ဖြစ်သည်။
ထပ်ဆင့်ပြီး၊ ယခုရေပန်းစားနေသော UI/UX ရဲ့ သဘောတရားများ၊ UX Design ဆိုတာဘာလဲ၊ UX Design ဆိုတာဘာလဲ၊ UI/UX designer ဖြစ်အောင် ဘယ်လိုလေ့ကျင့်ပြီး၊ ဘာတွေလေ့လာရမလဲ ဆိုသည်များကိုလည်း presnetation slide များဖြင့် ရှင်းလင်းသင်ပြသွားမည်ဖြစ်ပြီး၊ UX Designer တစ်ယောက် သိရမည့် အဓိက အချက်များ၊ UX Designer တစ်ယောက်ရဲ လုပ်ငန်းတာဝန်များ၊ UI Designer တစ်ယောက်သိထားရမည့် အဓိက အချ်ကများ၊ UI Designer တစ်ယောက်ရဲ လုပ်ငန်းတာဝန်များကို စာတွေ့ လက်တွေ့ သင်ကြားပြသသွားမည်ဖြစ်သည်။
လက်တွေ့အသားပေး သင်တန်းဖြစ်ပြီး သင်တန်းပြီးဆုံးပါက Professional Web Designer & Front-End Designer, UI/UX Designer တစ်ဦးအဖြစ် ရပ်တည်ရန် လိုအပ်သည့် ကျွမ်းကျင်မှုများကို ရရှိသွားမည် ဖြစ်သည်။
